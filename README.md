# 7S Rapport & Faltrapportverktyg

Webbaserade rapportverktyg för Hemvärnet och Försvarsmakten. Alla formulären är byggda som en PWA (Progressive Web App) och fungerar offline direkt i mobilen.

**Sajt:** [7srapport.com](https://7srapport.com)

## Bakgrund och Syfte
Detta projekt startade som ett försök att hitta ett snabbt och semiautomatiskt arbetsflöde för att effektivisera kommunikationen direkt till **Signal-appen** på telefonen ute i fält. Genom att använda färdiga mallar som genererar ren text, minimeras tiden för inmatning samtidigt som rapportformatet blir konsekvent och reglementsenligt.

Verktyget är en del i ett större pågående arbete med att digitalisera och förenkla ledningsstöd för svensk militär personal, med fokus på användarvänlighet under stress och i mörker... *TBC* :)

## Funktioner i urval
*   **7S-rapport:** Grundläggande spaningsrapport (Storlek, Slag, Sysselsättning...).
*   **Fordonsidentifiering:** Specialiserade formulär för stridsfordon (**WHAT**), flyg (**WEFT**) och civila fordon (**SCRIM**).
*   **Ledningsstöd:** Hjälpmedel för ordergivning (**OBO**), patrullverksamhet (**RASSOIKA**) och posttjänst (**SCHEMA**). *Notera: OBO och RASSOIKA är i ett tidigt utvecklingsstadium.*
*   **Anpassat för fält:** Mörkerläge (Dark Mode), stora knappar och PWA-stöd för 100% offline-användning.
*   **Metadata-stöttning:** Möjlighet att hämta tid och koordinater direkt från foton eller GPS.

## Verktyg

| Formulär | Beskrivning |
|----------|-------------|
| **7S** | Spaningsrapport (Storlek, Slag, Sysselsättning...) |
| **WHAT** | Stridsfordonsidentifiering |
| **SCRIM** | Civila fordon |
| **WEFT** | Flygfarkoster |
| **A-H** | Personbeskrivning / signalement |
| **OBSLÖSA** | Observationsrapport |
| **FORS** | Förbandsrapport |
| **PEDARS** | Stridsvärderapport |
| **SCHEMA** | Postschema med automatisk rotation |
| **EOBUSARE** | Eldorder |
| **OBO** | Orientering-Beslut-Order |
| **RASSOIKA** | Patrullchefens checklista |
| **VÄDER** | Meteorologisk prognos (SMHI-data) |
